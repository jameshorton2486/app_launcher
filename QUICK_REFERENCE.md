# Quick Reference Guide

Quick reference for common paths and configurations used in James's Project Launcher.

## External Tool Paths

Use these paths when configuring external tools in Settings:

| Tool | Path |
|------|------|
| **Cursor** | `C:\Users\james\AppData\Local\Programs\cursor\Cursor.exe` |
| **VS Code** | `C:\Users\james\AppData\Local\Programs\Microsoft VS Code\Code.exe` |
| **PyCharm** | `C:\Program Files\JetBrains\PyCharm Community Edition 2023.3.2\bin\pycharm64.exe` |
| **CCleaner** | `C:\Program Files\CCleaner\CCleaner64.exe` |
| **Wise Memory Optimizer** | `C:\Program Files (x86)\Wise\Wise Memory Optimizer\WiseMemoryOptimizer.exe` |

### Configuration in Settings

1. Open Settings (âš™ button in status bar)
2. Navigate to "EXTERNAL TOOLS" section
3. Click "Browse" next to each tool
4. Navigate to and select the executable path listed above

## Project Paths

Common project directories for quick reference:

| Project | Path |
|---------|------|
| **app_launcher** | `C:\Users\james\app_launcher` |
| **auto_capture_tool** | `C:\Users\james\auto_capture_tool` |
| **depo-pro** | `C:\Users\james\depo-pro` |
| **github_manager** | `C:\Users\james\github_manager` |
| **kollect-it** | `C:\Users\james\kollect-it` |
| **kollect-it-product-application** | `C:\Users\james\kollect-it-product-application` |
| **novel_assistant** | `C:\Users\james\novel_assistant` |

### Adding Projects

1. Go to Projects tab
2. Click "+ Add Project"
3. Enter project name
4. Set path to one of the paths above
5. Configure launch script and other details
6. Click "Save"

## Quick Setup Checklist

- [ ] Configure external tool paths in Settings
- [ ] Add all projects from the list above
- [ ] Set Downloads folder path
- [ ] Set Screenshots folder path (optional)
- [ ] Add Claude project URLs to projects

- [ ] Test git operations
- [ ] Test utilities

## Notes

- Paths may need to be updated if software is reinstalled or moved
- Claude project URLs should be updated when projects are created/accessed
- Project paths assume standard user directory structure
- Some paths may require administrator privileges to access
